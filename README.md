# PlayWithCaches

A chache and App data cleaner .

<h1> Download latest version via Gradle</h1>

<h3> STEP : 1 </h3>
    
    allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

<h3> STEP : 2 </h3>

    dependencies {
	        implementation 'com.github.sayyedrizwan:PlayWithCache:1.2.0'
	}
